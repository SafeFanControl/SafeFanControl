# SafeFanControl

**Mac의 팬을 직접 제어하세요**

SafeFanControl은 Apple Silicon 및 Intel Mac에서 온도를 모니터링하고 팬 속도를 관리할 수 있는 **네이티브 macOS 앱**입니다. 무료로 다운로드하세요 — 당신의 팬, 당신의 규칙. 문제가 생기면 시스템이 항상 제어권을 되찾습니다.

[macOS용 다운로드](https://github.com/SafeFanControl/SafeFanControl/releases)

---

## 기능

### 필요한 모든 팬 제어 모드

SafeFanControl은 작업 방식에 맞춘 세 가지 동작 모드로 Mac이 발열에 대응하는 방식을 완전히 제어합니다.

- **Monitor** — 시스템 제어에 개입하지 않고 온도와 팬 속도를 실시간으로 읽기
- **Safe Max** — 온도가 임계값에 도달하면 자동으로 최대 속도까지 올리고, 식으면 macOS에 제어권을 반환
- **Curve** — 연속적이고 비례적인 제어를 위한 사용자 정의 온도-속도 곡선 설정
- 메뉴 막대에서 언제든 모드 전환 — 앱 창 불필요

### 안전 우선 설계

3프로세스 아키텍처는 문제가 생기더라도 macOS가 자동으로 팬 제어권을 되찾도록 보장하며, 수동 개입이 전혀 필요하지 않습니다.

- **워치독 프로세스**가 헬퍼 데몬을 감시하여, 팬이 수동 모드인 상태에서 충돌하면 macOS가 즉시 제어를 넘겨받습니다
- **부드러운 페이드 전환**으로 소음이나 방해가 될 수 있는 급격한 팬 속도 변화를 방지합니다
- **임계값 래치**로 짧은 온도 변동 중에도 Safe Max를 안정적으로 유지합니다
- 표준 macOS 관리자 프롬프트로 백그라운드 헬퍼를 설치 — 터미널도, 수동 작업도 필요 없습니다

### 항상 메뉴 막대에

가벼운 메뉴 막대 앱이 실시간 센서 데이터와 모드 전환을 한 번의 클릭으로 제공합니다 — 항상 보이지만 결코 방해되지 않습니다.

- CPU 및 GPU 온도를 실시간으로 갱신
- 현재 팬 속도와 사용 중인 제어 모드를 항상 표시
- 드롭다운에서 Monitor, Safe Max, Curve를 바로 전환
- M1부터 M5까지의 Apple Silicon 및 T2 칩 탑재 Intel Mac 지원; 8개 언어 제공

## 다운로드 및 설치

### 시스템 요구 사항

| 요구 사항 | 세부 정보 |
|---|---|
| **macOS** | 15 (Sequoia) 이상 |
| **Mac 하드웨어** | Apple Silicon (M1, M2, M3, M4, M5) 또는 T2 칩이 탑재된 Intel Mac |
| **권한** | 백그라운드 헬퍼 데몬 설치를 위한 관리자 암호 입력 1회 |
| **디스크 공간** | 약 5 MB |

### 설치 방법

1. **다운로드한 `.dmg`를 열고** SafeFanControl을 응용 프로그램 폴더로 드래그합니다.
2. **앱을 실행합니다.** 첫 실행 시 관리자 대화 상자가 백그라운드 헬퍼와 워치독 데몬을 설치합니다.
3. **완료.** 터미널 불필요. 수동 승인 불필요. 팬 테스트 마법사가 사용자의 하드웨어에서 모든 것이 정상 작동하는지 확인하도록 안내합니다.

> 앱이 App Store를 통해 배포되지 않으므로 첫 실행 시 macOS가 Gatekeeper 경고를 표시할 수 있습니다. 계속하려면 **오른쪽 클릭 → 열기**를 선택하세요.

### 제거

SafeFanControl은 **설정 → 제거**에 완전한 제거 절차를 포함합니다. 헬퍼 데몬, 워치독 데몬 및 관련된 모든 파일을 한 번에 제거합니다.

## 자주 묻는 질문

**정식 기능에 대한 무료 체험이 있나요?**
네 — 첫 실행 시 Safe Max와 Curve 모드가 **14일 체험**으로 잠금 해제됩니다. 결제는 필요하지 않습니다.

**라이선스의 적용 범위는?**
하나의 라이선스로 당신의 모든 **개인용** Mac을 사용할 수 있습니다. 재배포나 상업적 대량 사용에는 사용할 수 없습니다.

**macOS 업그레이드 후 다시 구매해야 하나요?**
아니요. 라이선스는 영구적입니다.

**새 Mac을 구입하면 다시 구매해야 하나요?**
아니요 — 라이선스는 당신의 것입니다. 기존 라이선스 키로 새 Mac에 등록하세요.

**환불 정책은 어떻게 되나요?**
당신의 개인정보를 존중하기 때문에 이 앱은 오프라인 등록 시스템을 사용합니다. 라이선스 키는 한 번 발급되면 원격으로 비활성화할 수 없으므로, 모든 판매는 최종적이며 일반적인 환불은 불가능합니다. 구매 전 14일 무료 체험을 적극 권장합니다. 예외: Apple의 macOS 업데이트가 기반이 되는 팬 제어 방식을 영구적으로 비활성화하는 경우, 최근 30일 이내에 구매한 라이선스에 대해 전액 환불해 드립니다.

**왜 Mac App Store에 없나요?**
SafeFanControl은 권한이 있는 백그라운드 헬퍼 데몬을 설치하는데, 이는 App Store의 엄격한 샌드박스 규칙과 호환되지 않습니다. 직접 배포하지만, 설치 프로그램은 Apple의 서명 시스템을 통해 검사, 공증, 서명됩니다.

**결제하지 않고 사용할 수 있나요?**
네 — Monitor 모드는 완전히 무료이며 앞으로도 항상 무료입니다.

**향후 라이선스 서버가 오프라인이 되면 어떻게 되나요?**
라이선스 키는 오프라인으로 작동하며 인터넷 확인이 필요하지 않습니다. 라이선스 키가 있는 한 언제든지 등록할 수 있습니다.

**이 프로젝트를 포기하면 어떻게 되나요?**
모든 제품에는 수명이 있습니다. 그러나 제가 코딩할 수 있고 제품의 수익이 기본 비용(예: 연간 Apple Developer Program 비용)을 충당하는 한, 계속 유지보수하겠습니다. 이 제품이 마음에 드신다면 친구들에게 공유해 주세요. 만약(오직 그런 경우에만) 이 프로젝트를 포기해야 하는 날이 온다면, AGPL v3 라이선스에 따라 모든 유료 사용자에게 소스 코드를 공개하겠습니다.

**라이선스 키를 분실하면 어떻게 하나요?**
판매 플랫폼에 문의하시거나 저희에게 직접 연락해 주세요.
