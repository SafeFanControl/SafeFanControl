# SafeFanControl

**掌控你 Mac 的風扇**

SafeFanControl 是一款**原生 macOS 應用程式**，讓你監控溫度、管理 Apple Silicon 和 Intel Mac 的風扇轉速。免費下載——你的風扇，你做主，出現任何問題系統都會自動接管。

[下載 macOS 版本](https://github.com/SafeFanControl/SafeFanControl/releases)

---

## 功能特色

### 你所需的全部風扇控制模式

SafeFanControl 提供三種不同的運行模式，完全滿足你對 Mac 散熱控制的需求。

- **監控** — 即時讀取溫度和風扇轉速，不干預系統控制
- **安全最高速** — 當溫度超過門檻時自動提升至最高轉速，溫度降低後將控制權交還 macOS
- **曲線** — 自訂溫度-轉速曲線，實現連續比例控制
- 隨時從選單列切換模式，無需開啟應用程式視窗

### 安全優先的設計

三程序架構確保出現任何問題時，macOS 都能自動取回風扇控制權，無需任何手動干預。

- **看門狗程序**監控輔助常駐程式；若在手動模式下當機，macOS 將立即接管
- **平滑淡入淡出過渡**防止風扇轉速突變造成噪音或干擾
- **門檻鎖定**在短暫溫度波動時保持安全最高速穩定運行
- 標準 macOS 管理者提示安裝背景輔助程式——無需終端機，無需手動操作

### 始終常駐選單列

輕量級選單列應用程式讓即時感測器資料和模式切換只需一鍵——始終可見，從不妨礙使用。

- 即時 CPU 和 GPU 溫度更新
- 目前風扇轉速和使用中的控制模式始終可見
- 直接從下拉式選單在監控、安全最高速和曲線模式之間切換
- 支援 M1 至 M5 Apple Silicon 及搭載 T2 晶片的 Intel Mac；提供 8 種語言介面

## 下載與安裝

### 系統需求

| 需求 | 詳情 |
|---|---|
| **macOS** | 15 (Sequoia) 或更新版本 |
| **Mac 硬體** | Apple Silicon（M1、M2、M3、M4、M5）或搭載 T2 晶片的 Intel Mac |
| **權限** | 安裝背景輔助常駐程式需要一次管理者密碼提示 |
| **磁碟空間** | 約 5 MB |

### 安裝步驟

1. **開啟下載的 `.dmg`**，將 SafeFanControl 拖入「應用程式」資料夾。
2. **啟動應用程式。** 首次執行時，管理者對話框將安裝背景輔助程式和看門狗常駐程式。
3. **完成。** 無需終端機，無需手動操作。風扇測試精靈將引導你確認一切在你的硬體上正常運行。

> 由於應用程式不透過 App Store 分發，macOS 在首次啟動時可能顯示 Gatekeeper 提示。請**按住 Control 鍵點按 → 開啟**繼續。

### 解除安裝

SafeFanControl 在**設定 → 解除安裝**中提供完整解除安裝流程，一步移除輔助常駐程式、看門狗常駐程式及所有相關檔案。

## 常見問題

**有免費試用嗎？**
有——首次啟動後，安全最高速和曲線模式可免費試用 **14 天**，無需付款。

**授權涵蓋範圍是什麼？**
一個授權涵蓋你所有**個人** Mac，不可用於再分發或商業用途。

**macOS 升級後需要重新購買嗎？**
不需要，授權永久有效。

**更換新 Mac 需要重新購買嗎？**
不需要——授權屬於你個人。使用現有授權金鑰即可在新 Mac 上註冊。

**退款政策是什麼？**
由於我們尊重你的隱私，應用程式採用離線註冊系統。授權金鑰一經核發無法遠端停用，因此所有銷售均為最終銷售，不支援標準退款。我們強烈建議在購買前使用 14 天免費試用。例外：若 Apple macOS 更新永久停用了風扇控制所用的底層方法，我們將對過去 30 天內購買的授權給予全額退款。

**為什麼不在 Mac App Store 上架？**
SafeFanControl 需要安裝具特權的背景輔助常駐程式，與 App Store 嚴格的沙盒規則不相容，因此直接分發。但安裝包已透過 Apple 簽署系統進行掃描、公證和簽署。

**免費可以使用嗎？**
可以——監控模式完全免費且永久免費。

**授權伺服器將來下線怎麼辦？**
授權金鑰離線運作，無需連網驗證，只要你還有授權金鑰，隨時可註冊。

**如果你放棄這個專案怎麼辦？**
每個產品都有生命週期。但只要我還能寫程式，且產品收入能涵蓋基本開銷（如年度 Apple 開發者計畫費用），我就會持續維護。如果你喜歡這款產品，請分享給朋友。如果（僅當）有朝一日我不得不放棄這個專案，我將以 AGPL v3 授權向所有付費使用者開放原始碼。
